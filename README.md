# My MLB Passport

A small web app to track MLB games you’ve attended and explore “passport-style” stats (player leaderboards, team record when you attended, stadium counts, etc.).

## Stack
- Streamlit (frontend)
- Flask (REST API backend)
- MySQL (database)
- Docker Compose (local dev)

## Local Development

### Prerequisites
- Docker Desktop
- Git

### Run
From the repo root:

```bash
docker compose up --build
```

### Stop

```bash
docker compose down
```
